# Case p75 — supersonic flow with normal shock (pₑ/p₀ = 0.75)

At this back-pressure ratio the flow chokes at the throat, accelerates to supersonic in the diverging section, and a normal shock forms inside the diffuser. Downstream of the shock the flow returns to subsonic.

Initial pressure: p₀ = 10 kPa upstream of the throat; 7.5 kPa downstream, to accelerate convergence.
